## Description

This project features a layout designed with Tailwind CSS. The design aims to provide a clean and modern user interface, faithfully replicated from Figma.

## Design Choices

- **Direct Copy from Figma**: The design was copied directly from the Figma file, maintaining all visual elements and layout specifics.
- **No Specific Color Palette**: The colors used are directly taken from the Figma design.
- **No Custom Font Family**: The default Tailwind typography settings were used, matching the Figma design.

## Libraries Used

- **Tailwind CSS**: A utility-first CSS framework to build custom designs without writing custom CSS.

## Figma Design Reference

The layout was designed in Figma. You can view the design [here](https://www.figma.com/community/file/1145991068621514311).
